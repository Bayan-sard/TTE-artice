# Competing Risk Extension of the Cloning–Censoring–Weighting Framework

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18847191.svg)](https://doi.org/10.5281/zenodo.18847191)

This repository implements the cloning–censoring–weighting (CCW) framework as described by Maringe et al. (2020), *“Reflection on modern methods: trial emulation in the presence of immortal-time bias,”* International Journal of Epidemiology. The standard CCW components (cloning, artificial censoring, and inverse probability weighting) are adopted directly from the original work. The extension introduced in this repository consists of integrating competing risk estimation within the CCW framework to allow estimation of cumulative incidence in the presence of competing events.
